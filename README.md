# Proyek UTS Deep Learning

Proyek ini dibuat untuk memenuhi tugas Ujian Tengah Semester (UTS) mata kuliah Deep Learning. 
Fokus utamanya adalah memprediksi trajektori bola basket berdasarkan pergerakan dari frame-frame sebelumnya menggunakan model sequence (RNN, LSTM, GRU).

## Struktur Folder
- `data/` : Tempat menyimpan video raw dan dataset CSV.
- `notebooks/` : Tempat pengerjaan eksperimen model.
- `src/` : Kode sumber Python.
- `models/` : Model PyTorch yang telah di-training.
